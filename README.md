# PLY Music

我哒 🎓 设计作品 `^_^`

## 👀 特色功能

- 🎵 网易云、酷我、QQ 多播放源自动切换
- 🔍 搜索歌单
- ⏯ 播放动效
- 🎨 主题切换
- ❤️ 收藏音乐
- ⬇️ 下载音乐
- 💿 创建歌单
- 👩‍⚖️‍ 用户登录
